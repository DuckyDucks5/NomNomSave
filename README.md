## ğŸš€ Getting Started

### Prerequisites

- Flutter SDK
- Node.js and npm
- MySQL Server

### Backend Setup

1. Clone the repository
2. Navigate to the backend folder
3. Run:

```bash
npm install
npm start
```

4. Set up your `.env` file with your MySQL database credentials.

### Flutter App Setup

1. Navigate to the frontend folder
2. Run:

```bash
flutter pub get
flutter run
```

> Make sure a device or emulator is connected.
